ABOUT US

Chit Chat is a chat application designed for the simple purpose of interacting with people across the web. Still in the early stages, the application was designed with basic functionality where multiple sessions can connect to a hosted web server via an inputted username. Upon completion of a valid username, the session is established which then displays the next page where a list of connected users are present alongside an area for the users to message one another.

Chit Chat operates off of the NodeJS asynchronous, event driven framework which is able to handle simultaneous connections. ExpressJS, a web application framework that works in tandem with Node, handles the webpage routes such as the About Chit Chat page you are currently reading.

Chit Chat was created by a group of college students at William Paterson University located in Wayne, New Jersey. The application was constructed as a senior class project with a duration of less than a full semester (about three and a half months).
